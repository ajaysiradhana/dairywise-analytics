# DairyWise Analytics

A dairy management system built with Python, Flask, MySQL, and SQL.

## Features
- Customer Management
- Milk Records
- Payment Tracking
- Dashboard & Analytics

## Tech Stack
- Python
- Flask
- MySQL
- SQL
- HTML/CSS
